## 환불 정책 개요

CodeGen AI는 고객 만족을 최우선으로 합니다. 아래 정책에 따라 합리적인 환불을 지원합니다.

### 기본 원칙

- **전자상거래법 준수**: 청약철회 기간 내 환불 보장
- **투명한 정책**: 환불 조건 사전 고지
- **신속한 처리**: 환불 요청 후 영업일 기준 3-5일 내 처리

### 결제 유형별 요약

| 결제 유형 | 환불 가능 기간 | 환불 조건 |
|-----------|---------------|----------|
| 크레딧 구매 | 구매 후 7일 | 미사용 크레딧만 |
| 구독 (월간) | 결제 후 7일 | 미사용 시 전액, 사용 시 일할 계산 |
| 구독 (연간) | 결제 후 14일 | 미사용 시 전액, 사용 시 일할 계산 |

## 크레딧 환불

### 환불 가능 조건

1. **구매 후 7일 이내**
2. **크레딧 미사용**
   - 구매한 크레딧을 1회도 사용하지 않은 경우

### 환불 금액 계산

```
┌─────────────────────────────────────────────────────────────────┐
│                    크레딧 환불 금액 계산                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  미사용 (0회 사용) + 7일 이내                                     │
│  └─ 환불 금액 = 결제 금액 전액 (즉시 환불)                         │
│                                                                 │
│  일부 사용 (1회 이상 사용) + 7일 이내                              │
│  └─ 프로레이션 환불 (관리자 승인 필요)                             │
│     └─ 환불 금액 = 결제 금액 × (남은 크레딧 / 구매 크레딧)          │
│                                                                 │
│  7일 경과                                                        │
│  └─ 환불 요청 접수 (관리자 승인 필요)                              │
│     └─ 서비스 하자 또는 특수 사유 시 검토                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 프로레이션 환불 예시

크레딧 일부 사용 후 환불 요청 시:

| 상황 | 구매 | 사용 | 남은 크레딧 | 환불 금액 |
|------|------|------|-----------|----------|
| Standard 구매 후 30개 사용 | 150개 | 30개 | 120개 | ₩19,920 (24,900 × 120/150) |
| Premium 구매 후 100개 사용 | 350개 | 100개 | 250개 | ₩35,643 (49,900 × 250/350) |

> **참고**: 프로레이션 환불은 관리자 승인이 필요하며 영업일 기준 1-2일이 소요됩니다.

### 패키지별 환불 예시

| 패키지 | 결제 금액 | 미사용 시 환불 | 사용 후 환불 |
|--------|----------|---------------|-------------|
| Basic (50개) | ₩9,900 | ₩9,900 | 불가 |
| Standard (150개) | ₩24,900 | ₩24,900 | 불가 |
| Premium (350개) | ₩49,900 | ₩49,900 | 불가 |

### 크레딧 만료 관련

- 크레딧 만료로 인한 환불은 불가합니다.
- 만료 30일 전 알림을 발송합니다.
- 만료된 크레딧은 복구되지 않습니다.

## 구독 환불

### 월간 구독

#### 환불 가능 조건

1. **결제 후 7일 이내**
2. **청약철회 요청**

#### 환불 금액 계산

```
미사용 (서비스 이용 0일)
  └─ 환불 금액 = 결제 금액 전액

일부 사용 (1일 이상 이용)
  └─ 환불 금액 = 결제 금액 - (일일 요금 × 이용 일수) - 위약금(10%)

7일 경과 후 취소
  └─ 환불 없음, 기간 종료 시 자동 해지
```

#### 월간 구독 환불 예시 (Pro 플랜 ₩29,900)

| 상황 | 이용 일수 | 환불 금액 | 계산 |
|------|----------|----------|------|
| 당일 취소 | 0일 | ₩29,900 | 전액 환불 |
| 3일 후 취소 | 3일 | ₩23,932 | 29,900 - (997 × 3) - 2,990 |
| 7일 후 취소 | 7일 | ₩19,921 | 29,900 - (997 × 7) - 2,990 |
| 8일 이후 | - | ₩0 | 환불 불가 |

> **일일 요금 계산**: 월 요금 ÷ 30일 = ₩997/일

### 연간 구독

#### 환불 가능 조건

1. **결제 후 14일 이내** (전자상거래법 연간 계약 규정)
2. **청약철회 요청**

#### 환불 금액 계산

```
미사용 (서비스 이용 0일)
  └─ 환불 금액 = 결제 금액 전액

일부 사용 (1일 이상 이용)
  └─ 환불 금액 = 결제 금액 - (월 정가 × 이용 월수) - 위약금(10%)

14일 경과 후 취소
  └─ 환불 없음, 잔여 기간 이용 후 해지
```

#### 연간 구독 환불 예시 (Pro 플랜 연 ₩299,000)

| 상황 | 이용 기간 | 환불 금액 | 계산 |
|------|----------|----------|------|
| 당일 취소 | 0일 | ₩299,000 | 전액 환불 |
| 7일 후 취소 | 7일 | ₩239,200 | 299,000 - 29,900 - 29,900 |
| 14일 후 취소 | 14일 | ₩239,200 | 299,000 - 29,900 - 29,900 |
| 15일 이후 | - | ₩0 | 환불 불가 |

> **월 정가**: Pro ₩29,900, Team ₩99,000
> **연간 할인 포기**: 환불 시 월 정가 기준으로 이용 요금 계산

### 구독 취소 유형

| 취소 유형 | 설명 | 결과 |
|----------|------|------|
| **기간 종료 시 취소** | 현재 결제 기간 종료 후 해지 | 잔여 기간 이용 가능 |
| **즉시 취소** | 즉시 해지 (환불 대상) | 즉시 Starter로 변경 |

```typescript
// 기간 종료 시 취소 (권장)
await cancelSubscription({ cancelImmediately: false });

// 즉시 취소 (환불 요청 시)
await cancelSubscription({ cancelImmediately: true });
```

## 환불 절차

### 환불 유형별 절차

```
┌─────────────────────────────────────────────────────────────────┐
│                    환불 유형별 처리 절차                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [즉시 환불] 미사용 + 7일 이내                                    │
│  ├─ 설정 > 결제 관리 > 환불 신청                                 │
│  ├─ 조건 자동 검증 → 즉시 승인                                   │
│  └─ 토스페이먼츠 환불 API → 즉시 처리                             │
│                                                                 │
│  [관리자 승인 환불] 일부 사용 또는 7일 초과                         │
│  ├─ 설정 > 결제 관리 > 환불 요청                                 │
│  ├─ 환불 요청 생성 (status: pending)                            │
│  ├─ 관리자 검토 (1-2 영업일)                                     │
│  └─ 승인 시 프로레이션 금액 환불 / 거절 시 사유 안내               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Step 1: 환불 신청

1. **설정 > 결제 관리** 페이지 접속
2. 결제 내역에서 환불하려는 항목 선택
3. **환불 신청** 버튼 클릭
4. 환불 사유 선택 및 상세 내용 입력

### Step 2: 검토 및 승인

| 환불 유형 | 검토 방식 | 소요 시간 |
|----------|----------|----------|
| 즉시 환불 | 자동 검증 | 즉시 |
| 관리자 승인 | 담당자 수동 검토 | 1-2 영업일 |

### Step 3: 환불 처리

1. **카드 결제**: 카드사 취소 처리 (3-5 영업일)
2. **간편결제**: 해당 결제수단으로 환불 (1-3 영업일)

### 환불 요청 상태

| 상태 | 설명 |
|------|------|
| `pending` | 관리자 검토 대기 중 |
| `approved` | 승인 완료 - 환불 처리 중 |
| `rejected` | 거절됨 - 사유 확인 필요 |
| `canceled` | 사용자가 요청 취소함 |

### 환불 처리 타임라인

```
[즉시 환불]
Day 0: 환불 신청 → 즉시 승인 → 토스 환불 API 호출
Day 1-5: 카드사 처리 → 환불 완료

[관리자 승인 환불]
Day 0: 환불 요청 접수
Day 1-2: 관리자 검토 및 승인
Day 2-7: 카드사 처리 → 환불 완료
```

### 자동 처리 vs 수동 처리

| 구분 | 조건 | 처리 시간 | 환불 금액 |
|------|------|----------|----------|
| **즉시 환불** | 미사용 + 7일 이내 | 즉시 | 전액 |
| **프로레이션 환불** | 일부 사용 + 7일 이내 | 1-2 영업일 | 비례 계산 |
| **특수 환불** | 7일 초과 또는 서비스 하자 | 1-2 영업일 | 사안별 검토 |

### 환불 요청 취소

대기 중인 환불 요청은 언제든 취소할 수 있습니다.

1. **설정 > 결제 관리 > 환불 요청 목록** 접속
2. 대기 중(pending) 상태의 요청 선택
3. **요청 취소** 버튼 클릭

## 환불 불가 사유

다음의 경우 환불이 불가합니다:

### 크레딧

- 크레딧 1회 이상 사용한 경우
- 구매 후 7일 경과한 경우
- 크레딧 만료 후 요청한 경우
- 약관 위반으로 계정 정지된 경우

### 구독

- 청약철회 기간 경과 (월간 7일, 연간 14일)
- 무료 체험 기간 중 취소
- 약관 위반으로 서비스 제한된 경우

### 공통

- 허위 정보로 결제한 경우
- 부정 이용이 확인된 경우
- 환불 요청 후 계속 서비스 이용한 경우
