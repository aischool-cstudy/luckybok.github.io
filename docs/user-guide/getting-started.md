## CodeGen AI 소개

### CodeGen AI란?

CodeGen AI는 **AI 기반 코딩 교육 콘텐츠 자동 생성기**입니다.

- 🎯 **맞춤형 콘텐츠**: 학습자 수준에 맞는 교육 자료 생성
- ⚡ **빠른 생성**: 10분 안에 실무형 콘텐츠 완성
- 🌐 **다양한 언어**: Python, JavaScript, SQL 등 6개 언어 지원
- 📄 **PDF 내보내기**: 생성된 콘텐츠를 문서로 저장

### 누구를 위한 서비스인가요?

| 대상 | 활용 예시 |
|------|----------|
| 코딩 강사 | 강의 자료, 실습 문제 제작 |
| 기업 교육 담당자 | 사내 개발 교육 콘텐츠 |
| 부트캠프 운영자 | 커리큘럼 교재 개발 |
| 기술 블로거 | 튜토리얼 콘텐츠 작성 |

## 계정 생성

### 1단계: 회원가입 페이지 접속

홈페이지에서 **"무료로 시작하기"** 버튼을 클릭합니다.

```
URL: https://codegen.ai/register
```

### 2단계: 정보 입력

| 필드 | 설명 | 예시 |
|------|------|------|
| 이름 | 표시될 이름 | 홍길동 |
| 이메일 | 로그인 ID로 사용 | hong@example.com |
| 비밀번호 | 8자 이상, 영문+숫자 | MyPass123! |
| 비밀번호 확인 | 동일하게 입력 | MyPass123! |

### 3단계: 이메일 인증

1. 입력한 이메일로 인증 메일이 발송됩니다
2. 메일의 **"이메일 인증하기"** 버튼을 클릭합니다
3. 자동으로 로그인되어 대시보드로 이동합니다

> 💡 **팁**: 인증 메일이 안 보이면 스팸함을 확인해주세요.

## 첫 콘텐츠 생성하기

### 1단계: 생성 페이지 이동

1. 상단 메뉴에서 **"콘텐츠 생성"** 클릭
2. 또는 대시보드의 **"새 콘텐츠 만들기"** 버튼 클릭

### 2단계: 옵션 선택

#### 프로그래밍 언어

| 언어 | 추천 대상 |
|------|----------|
| 🐍 Python | 데이터 분석, 자동화, AI 입문 |
| 🌐 JavaScript | 웹 개발, 프론트엔드 |
| 📊 SQL | 데이터 조회, 분석 |
| ☕ Java | 백엔드, 엔터프라이즈 |
| 📘 TypeScript | 대규모 웹 애플리케이션 |
| 🚀 Go | 시스템 프로그래밍, 백엔드 |

#### 학습 주제

자유롭게 입력합니다. 구체적일수록 좋은 결과가 나옵니다.

```
✅ 좋은 예시:
- "파이썬 리스트 컴프리헨션"
- "JavaScript async/await 비동기 처리"
- "SQL JOIN 사용법과 실전 예제"

❌ 개선이 필요한 예시:
- "파이썬" (너무 광범위)
- "변수" (구체성 부족)
```

#### 난이도

| 난이도 | 설명 | 예상 학습 시간 |
|--------|------|--------------|
| 초급 | 기초 개념, 간단한 예제 | 15-30분 |
| 중급 | 실무 활용, 심화 개념 | 30-60분 |
| 고급 | 최적화, 아키텍처 패턴 | 60분+ |

#### 대상 학습자

| 대상 | 특징 |
|------|------|
| 비전공 직장인 | 업무 자동화 관점의 비유 |
| 주니어 개발자 | 기술적 깊이, 실무 패턴 |
| 관리자/임원 | 비즈니스 관점, 개념 중심 |
| 커리어 전환자 | 단계적 설명, 실습 중심 |

### 3단계: 콘텐츠 생성

1. 모든 옵션을 선택한 후 **"콘텐츠 생성"** 버튼 클릭
2. AI가 실시간으로 콘텐츠를 생성합니다
3. 생성 완료까지 약 30초~1분 소요

> ⚡ **실시간 스트리밍**: 콘텐츠가 생성되는 과정을 실시간으로 볼 수 있습니다.

## 결과 활용하기

### 콘텐츠 구조

생성된 콘텐츠는 다음 구조로 제공됩니다:

```
📄 생성된 콘텐츠
├── 📌 제목
├── 🎯 학습 목표 (3-5개)
├── 📖 핵심 개념 설명
│   └── 실생활 비유 포함
├── 💻 예제 코드
│   └── 주석 포함
├── ✏️ 연습 문제 (3개)
│   └── 정답 및 해설
└── 📋 핵심 요약
```

### 활용 방법

#### 1. 복사하기

- **전체 복사**: 상단의 "복사" 버튼 클릭
- **부분 복사**: 원하는 영역 드래그 후 복사

#### 2. PDF 내보내기

1. **"PDF 다운로드"** 버튼 클릭
2. 파일명 설정 (기본: 주제명_날짜.pdf)
3. 다운로드 완료

> 💡 **팁**: PDF에는 코드 하이라이팅이 포함됩니다.

#### 3. 히스토리 저장

- 생성된 모든 콘텐츠는 자동으로 저장됩니다
- **"히스토리"** 메뉴에서 이전 콘텐츠 확인 가능
- 최대 보관 기간: Starter 7일, Pro 30일, Team 무제한

### 콘텐츠 수정

생성된 콘텐츠는 마크다운 형식입니다. 필요에 따라 수정할 수 있습니다:

1. "편집 모드" 버튼 클릭
2. 내용 수정
3. "저장" 버튼 클릭

## 크레딧 시스템

### 크레딧이란?

콘텐츠 생성에 사용되는 포인트입니다.

| 플랜 | 무료 제공 | 충전 가능 |
|------|----------|----------|
| Starter | 일 10회 | ✅ |
| Pro | 일 100회 | ✅ |
| Team | 일 500회 | ✅ |

### 크레딧 사용량

| 작업 | 소모 크레딧 |
|------|-----------|
| 콘텐츠 생성 | 1 크레딧 |
| PDF 내보내기 | 무료 |
| 히스토리 조회 | 무료 |

### 크레딧 충전

추가 크레딧이 필요하면 충전할 수 있습니다.

| 패키지 | 크레딧 | 가격 | 크레딧당 |
|--------|--------|------|---------|
| Basic | 50 | ₩9,900 | ₩198 |
| Standard | 150 | ₩24,900 | ₩166 |
| Premium | 350 | ₩49,900 | ₩143 |

> 💰 **절약 팁**: Premium 패키지가 크레딧당 가격이 가장 저렴합니다.

### 크레딧 유효기간

- 구매 크레딧: 90일 (Premium은 180일)
- 무료 크레딧: 당일 자정 소멸

## 자주 묻는 질문

### 일반

<details>
<summary><strong>Q: 생성된 콘텐츠의 저작권은 누구에게 있나요?</strong></summary>

생성된 콘텐츠의 저작권은 사용자에게 있습니다. 상업적 용도로 자유롭게 사용할 수 있습니다.
</details>

<details>
<summary><strong>Q: 콘텐츠 품질이 마음에 들지 않으면 어떻게 하나요?</strong></summary>

1. 주제를 더 구체적으로 입력해보세요
2. 대상 학습자를 정확히 선택해주세요
3. 동일 주제로 다시 생성하면 다른 결과가 나옵니다
</details>

<details>
<summary><strong>Q: 오프라인에서도 사용할 수 있나요?</strong></summary>

콘텐츠 생성은 인터넷 연결이 필요합니다. 하지만 생성된 콘텐츠는 PDF로 다운로드하여 오프라인에서 사용할 수 있습니다.
</details>

### 결제

<details>
<summary><strong>Q: 환불이 가능한가요?</strong></summary>

- 구독: 7일 이내 미사용 시 전액 환불
- 크레딧: 미사용 크레딧에 대해 환불 가능

자세한 내용은 [환불 정책](../refund-policy.md)을 참조하세요.
</details>

<details>
<summary><strong>Q: 결제 수단은 무엇이 있나요?</strong></summary>

- 신용카드/체크카드
- 간편결제 (토스페이, 네이버페이 등)
- 가상계좌
</details>

<details>
<summary><strong>Q: 구독을 취소하면 어떻게 되나요?</strong></summary>

- 현재 결제 기간 종료까지 서비스 이용 가능
- 기간 종료 후 Starter(무료) 플랜으로 자동 전환
- 히스토리는 7일간 유지 후 삭제
</details>

### 기술

Anthropic의 Claude AI를 기본으로 사용합니다. 최신 AI 기술로 고품질 콘텐츠를 생성합니다.
</details>

<details>
<summary><strong>Q: API로 연동할 수 있나요?</strong></summary>

Team 플랜 이상에서 API 액세스를 제공합니다. API 문서는 설정 > API 키 메뉴에서 확인할 수 있습니다.
